# Sensei

## Overview

Sensei is an AI-powered voice assistant designed to provide natural, conversational interactions through voice.

It listens to the user, understands requests using Google's Gemini AI, and responds with both text and speech.

The project emphasizes a clean user experience, modular architecture, and real-time voice interaction.

---

## Objectives

- Natural voice conversation
- Fast AI responses
- Beautiful modern interface
- Real-time speech recognition
- Voice synthesis
- Conversation memory
